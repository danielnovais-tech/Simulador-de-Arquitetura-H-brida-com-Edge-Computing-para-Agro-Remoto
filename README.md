# Simulador-de-Arquitetura-H-brida-com-Edge-Computing-para-Agro-Remoto
Simula rede híbrida, edge computing resiliente e testes de validação

## Dicas para rodar e observar melhor

A saída do simulador é verbosa → rode em um terminal grande ou redirecione para arquivo se quiser analisar depois:

```bash
python agro_edge_simulator.py > simulacao_$(date +%Y%m%d_%H%M).log
```

Isso criará um arquivo de log com timestamp (ex: `simulacao_20231229_1630.log`) para análise posterior.
